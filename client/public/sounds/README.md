# 🎵 مجلد الأصوات

هذا المجلد يحتوي على ملفات الأصوات المخصصة للتطبيق.

## 📁 الهيكل

```
sounds/
├── letters/        ← أصوات الحروف الفردية (أ.mp3, ب.mp3, إلخ)
├── words/          ← أصوات الكلمات (كتاب.mp3, قلم.mp3, إلخ)
├── ui/             ← أصوات الواجهة (success.mp3, error.mp3, إلخ)
└── songs/          ← الأغاني والموسيقى (alphabet.mp3, إلخ)
```

## 📝 ملاحظات

- جميع الملفات يجب أن تكون بصيغة MP3
- استخدم أسماء الملفات بالعربية للحروف والكلمات
- تأكد من أن أسماء الملفات تطابق بالضبط الحروف/الكلمات المستخدمة في الكود
