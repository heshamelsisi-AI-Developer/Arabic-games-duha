### Slide 4: Game 1 - Flashcards (Part 1)
**Title:** Flashcards Game: How It Works

**Content:**
The flashcards game introduces children to letter sounds through audio-visual interaction.

**Mechanics:**
1. App plays audio pronunciation of an Arabic letter
2. Child sees 4 colorful card options
3. Child clicks the correct letter
4. Instant visual feedback (green bounce for correct, red shake for incorrect)
5. Score automatically updates

**Benefits:**
- Auditory learning: Children hear correct pronunciation
- Visual learning: Colorful cards maintain attention
- Immediate feedback: Children know instantly if they're correct
- Self-paced: No pressure to rush

**Visual:** Screenshot of flashcards game interface

### Slide 6: Game 2 - Memory (Part 1)
**Title:** Memory Game: How It Works

**Content:**
The memory game reinforces letter recognition through visual matching and pattern recognition.

**Mechanics:**
1. 8 cards displayed face-down (4 pairs)
2. Child flips cards to reveal letters
3. Goal: Find matching letter pairs
4. Matched pairs stay revealed
5. Unmatched cards flip back after 0.8 seconds
6. Game ends when all pairs are matched

**Benefits:**
- Visual recognition: Children learn letter shapes
- Memory development: Improves working memory
- Pattern recognition: Develops cognitive skills
- Engagement: Game-like mechanics maintain interest

**Visual:** Screenshot of memory game at different stages

### Slide 10: Classroom Integration
**Title:** Instructor Tips & Best Practices

**Content:**
Practical guidance for incorporating games into teaching sessions.

**Best Practices:**
- **Sequencing:** Start with Flashcards to introduce new letters, then use Memory for reinforcement
- **Pacing:** Allow children time to think; don't rush them
- **Pronunciation:** Encourage children to repeat letter sounds aloud
- **Celebration:** Enthusiastically celebrate correct answers to maintain motivation
- **Variety:** Rotate between games to maintain engagement
- **Group Play:** Use on classroom projector for group learning sessions

**Session Structure:**
1. Warm-up: 5 minutes of Flashcards (familiar letters)
2. Learning: 10 minutes of Flashcards (new letters)
3. Practice: 10 minutes of Memory (mixed difficulty)
4. Celebration: Acknowledge progress and achievements
